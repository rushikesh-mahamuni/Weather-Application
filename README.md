
# Weather Application






## Installation

Install this Project with npm

```bash
  npm install
  
```


## API Reference

#### Call current weather data for one Location by City Name & Country Name :

```http
  "http://api.openweathermap.org/data/2.5/weather?q=${city},${country}&appid=${Api_Key}"
```






##  Tech Stacks

### React Js

    React helps you create your web applications in a more maintainable way.
    React enables developer to reuse components.  

### Bootstrap
    
#### Install bootstrap using :

```bash
  npm install bootstrap
```
#### Import CSS using :

```bash
  import "bootstrap/dist/css/bootstrap.min.css"
```
### Weather Icons

#### Install Weather Icons using :

```bash
  npm install https://github.com/erikflowers/weather-icons.git 
```
#### Import CSS using :

```bash
import "weather-icons/css/weather-icons.css"  
```
